# Adib's Personal Portfolio 🚀

Welcome to the source code for my personal portfolio website. I am a Cybersecurity student , passionate about keeping the internet safe, building stuff, and breaking things (legally 😄). 

This repository serves as a showcase of my front-end development skills, campus initiatives, and technical projects.

## 🌐 Live Preview
Check out the live website here: [https://SalmonIsFish.github.io/portfolio](https://SalmonIsFish.github.io/portfolio)

## 🛠️ Tech Stack
This project was built from scratch without any heavy frameworks to practice core web development fundamentals:
* **HTML5** for semantic structure
* **CSS3** for styling, responsive grid/flexbox layouts, and custom animations
* **JavaScript (Vanilla)** for interactive DOM manipulation and canvas rendering

## ✨ Key Features
* **Dynamic UI Elements:** Includes a custom typewriter effect and floating background particles.
* **Activities & Events Timeline:** Highlighting my involvement with the CLICQ IT Society, organizing the Setel Industrial Visit, and hosting the AquFish Technology agrotech startup talk.
* **Interactive Modals:** Custom-built pop-ups for clean user interaction.
* **Hidden Easter Egg:** A fully functional 2D space shooter mini-game built entirely with the HTML5 Canvas API. Score 20 points to unlock my contact info! 👾
